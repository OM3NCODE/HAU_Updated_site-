1. PROJECT OVERVIEW & GOALS
What Is This Site?
highasunicorn.meme is the main website for High As Unicorn (HAU), a dual-chain Web3 community project with two meme tokens ($HAU and $UBP) and one NFT collection (Hauwee). The site replaces whatever existed before and becomes the single source of truth for the project.
What It Is NOT
NOT a DApp — no wallet connections, no on-chain transactions on the site itself
NOT a marketplace — minting links to LaunchMyNFT, trading links to Magic Eden
NOT where the game lives — Hauwee’s Lava Run is on Discord/Telegram only
NOT shithappens.com — that’s a separate landing page (redirects to DexScreener)

2. USER JOURNEY & CONVERSION FLOW
The Core Loop
Every page on the site feeds one loop: visitor discovers HAU → shill math hooks them → they mint a Hauwee → they join Discord → they play the game → they earn $UBP → they buy more $HAU → NFT value rises → more people mint. The site exists to start this loop.
Journey Map: New Visitor (Degen from X)
Lands on home page from X link or shithappens.com redirect
Sees hero: "Two Tokens. One NFT. Two Chains." — understands scope instantly
Scrolls to ecosystem cards: $HAU, $UBP, Hauwee — clicks whichever interests them
Hits shill math banner: "Mint at 0.22, floor at 0.4" — the hook
Clicks "Mint a Hauwee" → NFT page → sees progress bar (906/1,111) → mints on LaunchMyNFT
Returns, sees "Join Discord for game nights" → joins Discord
Discovers Hauwee’s Lava Run in Discord → plays → earns $UBP
Visits token page to buy more $HAU and $UBP on Raydium/DexScreener
Becomes community member → attends Friday Spaces → tells friends
Journey Map: Existing Holder (Checking In)
Lands on home page — sees updated stats (minted count, floor price)
Checks token page for live DexScreener charts
Visits game page for leaderboard / game night schedule
Shares site link with friends as a shill tool
KEY DESIGN PRINCIPLE
Every page must have at least TWO visible CTAs at all times: (1) Mint a Hauwee (primary, in nav), and (2) Join Discord (secondary). The token CAs must be visible in the footer without scrolling on desktop. On mobile, a sticky "Mint" button at the bottom of the screen.

Typography
Display / Headlines: Use a bold, characterful font. Suggestions: Satoshi, General Sans, or Cabinet Grotesk (all free on Fontshare). NOT Inter, Roboto, or Arial. The font should feel degen-friendly but readable.
Body Text: Pair with a clean sans-serif. DM Sans, Plus Jakarta Sans, or the same display font at lighter weights.
Monospace (for CAs): JetBrains Mono or Fira Code — contract addresses must be scannable and copy-friendly.
Size Scale: Hero headline: 48–56px. Section titles: 28–32px. Body: 16–18px. Captions: 12–14px.
Tone & Vibe
THE FEEL
Degen-but-wholesome. Playful, not corporate. Confident, not try-hard. The site should feel like a friend who’s good at crypto inviting you in — not a pitch deck from a startup. Think: Pudgy Penguins energy meets community-first messaging.
DO: Use memes, humor, the Hauwee character, casual language, real numbers
DO: Show the shill math prominently — it’s the hook
DO: Use “Hau ya doing?” as the greeting throughout
DON’T: Corporate jargon, stock photos, stiff language
DON’T: Promise returns, use rocket emojis, or say “to the moon”
DON’T: Make it look like every other Solana meme site (gradient purple on black)
The Hauwee character should appear throughout the site in different poses/traits. Sean’s hand-drawn style IS the brand — don’t polish it into something generic. The charm is in the goofy, hand-drawn, derpy energy.
